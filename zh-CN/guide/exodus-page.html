# εxodus 页

εxodus page appears after clicking on the εxodus button in the App Info tab in the App Details page. It scans for trackers and displays a list of trackers and their classes as a summary. A list of tracker classes for the given app is also displayed. All the classes of the app can also be viewed by clicking on the Toggle Class Listing menu. A sneak-peek of each class can be viewed by simply clicking on any class item. The tracker names have some prefixes, and although they are not properly documented in the original repository (opens new window), a list is given below:

  • ° - Missing in εxodus' list
  • ² - ETIP (opens new window) stand-by
  • µ - Micro non-intrusive
  • ? - Unconfirmed
